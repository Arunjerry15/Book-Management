WELCOME TO ARUN BOOK STORE

This is my Book Management Project is a web-based application designed to manage a collection of books efficiently. Users can add new books by providing details like the book's name, author, and price. The project allows users to edit book details to keep information updated. Additionally, books can be deleted from the list when no longer needed. A special feature enables users to mark books as favorites, adding them to a personalized list for easy access. The project features a clean user interface with navigation links for smooth browsing. The home page displays available books, while the favorite books section keeps track of personal choices. It leverages Spring Boot for backend operations and Thymeleaf for dynamic front-end rendering. The database is managed using MySQL, ensuring data integrity and persistence. With intuitive CRUD functionality, this project simplifies book collection management efficiently.
